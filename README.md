haushaltsdatenvergleich
=======================

Haushaltsdatenvergleich NRW


Projekt kompilieren
-------------------
Im Verzeichnis "converter" 

mvn package 

ausführen.


Daten konvertieren
------------------

Im Projekt-Verzeichnis die Skripte "konvertiere_gemeinden" und "konvertiere_landkreise" ausführen.

Die konvertierten Daten werden gleich im Verzeichnis viewer\web\js erstellt.
